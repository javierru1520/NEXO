🚌 Traxión Academia del Operador — Guía de integración

v1.0 Single-file HTML Sin dependencias locales · Funciona offline

TL;DR: La forma más rápida y compatible es el iframe. Una sola línea y funciona en cualquier sistema.

Opción 1 — iframe Recomendado

El más compatible. Funciona en cualquier CMS, portal, Webflow, SharePoint, etc.

Básico

<iframe
  src="./traxion_lms/index.html"
  style="width:100%; height:100vh; border:none; display:block;"
  title="Traxión Academia del Operador"
></iframe>

Responsivo con altura mínima

<div style="position:relative; width:100%; min-height:700px;">
  <iframe
    src="./traxion_lms/index.html"
    style="position:absolute; inset:0; width:100%; height:100%; border:none;"
    allowfullscreen
  ></iframe>
</div>

Abrir directamente en la app del operador

<!-- Agregar ?page=app-operador para iniciar en la app móvil -->
<iframe src="./traxion_lms/index.html#app-operador" ...></iframe>

Opción 2 — embed.js Avanzado

Inyecta el LMS dentro de un div en tu página existente.

HTML

<!-- 1. Crea el contenedor -->
<div id="traxion-lms" style="width:100%; height:100vh;"></div>

<!-- 2. Carga el script con el selector del contenedor -->
<script
  src="./traxion_lms/assets/js/embed.js"
  data-target="#traxion-lms"
  data-mode="hr"
></script>

JavaScript manual

// Llamada manual con opciones
TraxionLMS.mount('#traxion-lms', { mode: 'iframe' });

// Navegar a una sección después de montar
TraxionLMS.navigate('app-operador');  // abre la app del operador
TraxionLMS.navigate('alertas');       // abre alertas
TraxionLMS.navigate('dashboard');     // dashboard RRHH

Notas técnicas

  • El archivo index.html es 100% standalone — CSS, JS y datos están inline. Se puede usar solo sin ningún otro archivo.
  • La carpeta assets/ es para integración avanzada donde se prefiere separar capas.
  • La única dependencia externa es Google Fonts (Barlow) — si hay red, se carga automáticamente. Sin red, usa la fuente del sistema.
  • Compatible con Chrome 90+, Edge 90+, Safari 14+, Firefox 88+.
  • Sin cookies, sin localStorage, sin tracking. Todos los datos son en memoria (dummy).
